# Обложки новостей (опционально)

Пока новости показываются с фирменным плейсхолдером — сайт выглядит цельно и без фото.

Чтобы добавить обложку к новости:
1. Положи изображение сюда, например `public/news/riyadh-space-lab.jpg`
   (лучше горизонтальное, ~1600×1000, .jpg/.webp).
2. Открой `lib/news.ts`, найди нужную новость по `slug` и добавь строку:
   `image: "/news/riyadh-space-lab.jpg",`
3. Всё. Если поля `image` нет — автоматически рисуется плейсхолдер.

Соответствие slug → событие см. в `lib/news.ts` (id 31–41).
